This page uses the open-source Neural Amp Modeler WebAssembly player from TONE3000. It can load a NAM amp model, a cab IR, and a dry guitar input so you can test IRs through a real NAM-style model instead of the simple built-in amp sim.

For best results, use a short dry DI guitar clip and a normal mono cab IR. The normal mixer page is still the one that renders and downloads your blended IR file.

Important

This is an experimental page. It relies on a WebAssembly package loaded from a CDN, so it needs internet access and may take a little longer to start. If it does not load on GitHub Pages, the original IR mixer still works locally without NAM.

How this fits the IR mixer

  • Use this page to hear how an IR sounds through an actual NAM model.
  • Use the main mixer page to blend two IRs and download the mixed IR.
  • After downloading the mixed IR, load that WAV here as the cab IR.

Good test workflow

  • Pick a NAM model close to your normal amp tone.
  • Load a dry DI riff that has palm mutes, chords, and single notes.
  • Compare IR blends at the same output level so louder does not automatically seem better.
