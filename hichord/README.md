# HiChord iPhone PWA

This folder contains the iPhone-ready HiChord web app.

## Use on iPhone

1. Open the hosted page in Safari.
2. Tap the Share button.
3. Tap **Add to Home Screen**.
4. Launch HiChord from the Home Screen and rotate the phone sideways.

The app uses Web Audio, touch controls, a web manifest, and a service worker for offline use.
